# Brick Breaker 2D


## Description

Brick Breaker 2D is a classic arcade game in which the player must smash a stack of coins resembling bitcoins by deflecting a bouncing ball with a paddle. It's a fun and addictive game that tests your reflexes and aiming skills.

## Gameplay

- Control the paddle using the left and right arrow keys on your keyboard 
- Bounce the ball off the paddle to break the bricks.
- Try to clear all the bricks to advance to the next level.

## Technologies Used

- HTML5
- CSS3
- JavaScript
